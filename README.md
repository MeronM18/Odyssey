A travel planning web application for booking flights, hotels, events, and car rentals.

## Tech Stack
- Node.js + Express (backend)
- Supabase (database + auth)
- PostgreSQL
- HTML, CSS, JavaScript (frontend)

## Run the Project

**Windows:**
```bash
.\start.bat
```

**Manual:**
```bash
cd Odyessia/backend
npm run dev
```
```bash
cd Odyessia/frontend
npx serve -l 3000
```

Then open: **http://localhost:3000/html/**

## Features
- Google OAuth authentication
- Flight, hotel, event, and car rental bookings
- User profiles with budget tracking
- Row Level Security for data protection
